# 13. 퀴즈 시스템 기능 구현

## 개요
AI 기반 퀴즈 생성 시스템의 핵심 백엔드 기능을 구현합니다. OpenAI API를 활용한 문제 자동 생성, 퀴즈 실행 로직, 채점 시스템, 그리고 학습 진도 추적 시스템을 완성합니다.

## 목표
- AI 기반 문제 자동 생성 시스템 구현
- 다양한 문제 유형별 생성 및 채점 로직
- 퀴즈 실행 및 상태 관리 시스템
- 학습 분석 및 진도 추적 기능
